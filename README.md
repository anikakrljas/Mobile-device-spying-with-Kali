## üß™ Step 1: Installing Genymotion

### ü§ñ What is Genymotion?

**Genymotion** is a powerful Android emulator used by developers and security researchers to run Android devices in a **virtual, controlled environment**.  
Instead of testing on a real phone, Genymotion allows you to safely simulate different Android versions and device models on your computer.

This makes it ideal for:
- Security research üîê  
- Application testing üß©  
- Learning how mobile systems work in isolation üß†  

> üß† Using emulators is a best practice when learning mobile security ‚Äî no real devices are put at risk.

## üßæ Step 2: Creating a Genymotion Account

To use Genymotion, you must create a **free user account**.

### Steps:
1. Open **Genymotion Desktop**
2. Click **Sign in** ‚Üí **Create an account**
3. Enter your email and password
4. Confirm your email via the verification link
5. Log in and choose the **Personal Use** license

![Create account](https://github.com/user-attachments/assets/11dd32b3-52e5-4c30-9ca5-4ba90f9f8b48)

After logging in, you can start downloading and running virtual Android devices ‚úÖ

## üì≤ Step 3: Creating a Virtual Android Device

After logging in, you can create your first virtual device.

### Steps:
1. In **Genymotion Desktop**, click on **Create Device**
2. Choose **any device model** (the specific model is not important)
3. Leave all other settings on their **default values**
4. Click **Create** and wait for the setup to finish

Once the process is complete, your virtual Android device will appear in the list and is ready to be started üöÄ

![Create device](https://github.com/user-attachments/assets/08f49958-6c51-4d03-ad5d-bc4310a72ac3)

Once the device is created, it is launched inside **VirtualBox**.  
Simply click the **Play ‚ñ∂Ô∏è icon** next to the device in **Genymotion**, and the Android emulator will start automatically.
